# Learning Log

Document your learnings, experiments, and insights here.

## Format
Use weekly logs: `YYYY-MM-weekN.md`

Example: `2025-10-week1.md`

## Template
```markdown
# Week N - YYYY-MM-DD to YYYY-MM-DD

## What I Learned
- Topic 1
- Topic 2

## Experiments
- What I tried
- Results

## Challenges
- Problem encountered
- Solution found

## Resources
- Links to helpful articles
- Documentation references

## Next Week Goals
- [ ] Goal 1
- [ ] Goal 2
```
